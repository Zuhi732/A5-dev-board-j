Question-1: What are the different ways to select an element in the DOM?

We can select elements using:

  • document.getElementById("id") - Selects an element by ID.
  • document.getElementsByClassName("class") - Selects elements by class name.
  • document.getElementsByTagName("tag") - Selects elements by tag name.
  • document.querySelector("selector") - Selects the first matching element.
  • document.querySelectorAll("selector") - Selects all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Returns or sets the HTML content inside an element.
innerText: Returns or sets only the visible text inside an element.
textContent: Returns or sets all text, including hidden content.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for multiple child elements. Instead of adding event listeners to individual elements, we attach one listener to a common parent.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event on a child element automatically moves up (or "bubbles") to its parent elements.
Example: If we click a button inside a <div>, the click event will also trigger on the parent <div>.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating an Element: Use document.createElement("tag") to create a new element.
Adding an Element: Use parent.appendChild(newElement) to insert it into the page.
Removing an Element: Use element.remove() to delete it from the page.
